Reverse every natural instinct that you possess and do the opposite of what you are inclined to do everytime and you will probably come very close to having a perfect coding swing.This is the first time that we are going to organize such an event. This event is set up in 2 portfolios : In the first section several compiled code are given and on the other hand you have to code the reverse logic of given codes. Whosoever winds up first with maximum score obtained in passing the test cases will breach the boundaries of their techinal skills.You can code in any language that you want to .So be ready to code in reverse world.

So brush up your basics and logics and buckle up to code and find a way back home.
